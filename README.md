# William-s-Weather-Station-2025-2026-for-HGS
This is for people who are trying to code components such as the BME280, PMS5003 and DS18B20

This code is for any project which uses these components: PMS5003, BME280 and DS18B20. Make sure to import the modules before running the code, or else it could crash. Further, actual finalised code coming soon! :)

An overview of what has happened so far...

File Name: FinalWeatherStationGUI.py - this file is a simulation for the BME280, PMS5003 and DS18B20. The numbers generated are not real, but are in fact random. The GUI comes with accessibility features such as High-Contrast Mode and Zoom.

File Name: WeatherMasterPro - this is the final code; feel free to make changes!

File Name: BME280 - this file is just a testing file for the BME280 component

File Name: PMS5003 - this file is just a testing file for the PMS5003 component

File Name: DS18B20 - this file is just a testing file for the DS18B20 component
